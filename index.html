This website is in construction.

It will contain simulations of control systems and animations of important concepts from perceptual control theory (PCT). PCT suggests a unique and interesting way of applying classical control theory to experimental psychology, as well as constructing artificial control systems. It was developed by the late William T. Powers.

My first project is making JavaScript versions of simulations written in Delphi by Powers and Abbott, and published with the book Living Control Systems III. Delphi source code with compiled programs is available at: www.billpct.org
